Project Vision

Build an “E-Commerce Sales Analytics Platform” that unifies batch, ETL, SQL analytics, and real-time processing to generate business insights and dashboards.

  • Process large-scale transaction data (CSV/JSON)
  • Integrate MapReduce, Pig, Hive, and Spark workloads
  • Deliver visualizations and executive reporting

Architecture Blueprint

The layered architecture guides data from ingestion to actionable insights, with explicit tooling decisions for each stage.

Data Sources → HDFS MapReduce (Batch KPIs) Pig (ETL) Hive (SQL Warehouse) Spark (Real-time) Visualizations & Dashboards

Phase 2–3

MapReduce analytics and Pig ETL pipelines.

Phase 4

Hive warehouse, table formats, partitions, UDFs.

Phase 5–6

Advanced analytics, visualization, and reporting.
